## Where to source

- Manufacturer press kits (Ohme, Hypervolt etc. publish marketing assets)
- Distributor catalogues (with permission)
- Your own product photography from past installs

## Fallback

If an image is missing, the site renders a clean SVG illustration in its place — no broken-image icons appear. So the site works fine before you've added anything; just gets richer once images are dropped in.
